**文档版本**：1.3（2026-03-20）

## 1. 你可以先知道的 3 件事

1. 这个程序主要用于课堂上的点名、计时、批注和演示，不是教务系统。
2. 大多数功能都从桌面上的悬浮启动器进入，不需要反复找菜单。
3. 程序默认使用本地文件，学生名册和照片建议保存在学校电脑的固定位置。

## 2. 安装与第一次启动

### 如果你拿到的是成品包

1. 将程序文件夹解压到一个固定位置
2. 双击 `sciman Classroom Toolkit.exe`
3. 稍等片刻，屏幕边缘会出现一个小圆形悬浮按钮

这个小圆形按钮就是启动器，后续点名、画笔、图片和 PDF 等功能都从这里进入。

### 第一次启动后会发生什么

- 程序会自动准备必要配置
- 如果还没有学生名册，点名功能会提示你准备模板
- 已有学生数据时，可以直接进入点名窗口使用

### 如果启动失败

常见情况：

- 系统提示缺少 `.NET` 运行时
- 双击后没有反应
- 被学校电脑的安全策略拦截

处理建议：

- 先重新双击一次，确认不是误操作
- 如果仍然无法启动，请联系学校信息老师检查运行环境和权限

> 一般教师不需要自己安装运行环境。学校统一部署时，由信息老师处理更稳妥。

## 3. 课前 3 分钟检查清单

正式上课前，建议快速检查以下内容：

1. 打开程序，确认悬浮启动器已经出现
2. 打开点名窗口，确认班级和学生能正常显示
3. 随机点一次名，确认是否有语音播报
4. 打开一张图片或一个 PDF，确认能翻页和批注
5. 如果本节课要用 PPT / WPS，先试一次放映和滚轮翻页

如果这 5 项都正常，课堂中出问题的概率会明显降低。

## 4. 准备学生名册与照片

### 学生名册 `students.xlsx`

学生名册按“一个工作表对应一个班级”来管理。

建议保留这些字段：

- `学号`
- `姓名`
- `分组`（可选）

示例：

```text
students.xlsx
├── 1班
│   学号   姓名   分组
│   001    张三   第一组
│   002    李四   第二组
└── 2班
    学号   姓名   分组
    101    王五   第一组
```

使用建议：

- 一个班级一个工作表
- 工作表名就是班级名，例如 `1班`、`2班`
- `学号` 和 `姓名` 建议都填写
- `分组` 可以不填，但分组点名时会更方便
- 如果找不到 `students.xlsx`，程序通常会自动生成模板

### 学生照片 `student_photos/`

照片目录建议按“班级文件夹 + 学号文件名”存放：

```text
student_photos/
├── 1班/
│   ├── 001.jpg
│   └── 002.png
└── 2班/
    └── 101.jpg
```

要点：

- 班级文件夹名要和工作表名一致
- 照片文件名建议使用学号，不建议直接用姓名
- 支持 `.jpg`、`.jpeg`、`.png`、`.bmp`
- 没有照片也可以点名，只是不会显示头像

## 5. 启动器：所有功能的入口

启动程序后，屏幕边缘会出现一个小圆形悬浮按钮。

常见操作：

| 操作 | 结果 |
|------|------|
| 单击悬浮按钮 | 展开功能菜单 |
| 选择「点名 / 计时」 | 打开点名窗口 |
| 选择「画笔 / 白板」 | 打开画笔叠加层 |
| 选择「打开图片」 | 打开图片 / PDF 浏览 |
| 双击悬浮按钮 | 收起菜单 |
| 拖拽悬浮按钮 | 移动到屏幕其他位置 |

使用建议：

- 把启动器拖到自己顺手的位置
- 如果课堂中不想遮挡内容，可以移动到屏幕边角
- 如果启动器消失，先回到桌面找一下，必要时重启程序

## 6. 随机点名与计时器

### 开始点名

1. 点击启动器，选择「随机点名」
2. 打开点名窗口后，按 `空格` 或点击窗口
3. 程序会显示被抽中的学生姓名、学号，若有照片会一起显示
4. 再按一次 `空格`，继续下一位

### 切换班级和分组

- 窗口顶部可以切换班级
- 可以按分组范围点名，例如“全部”“第一组”“第二组”

### 计时器怎么用

点名窗口中通常可以切换计时器模式。

| 功能 | 说明 |
|------|------|
| 倒计时 | 输入分钟数，时间到后播放提示音 |
| 正计时 | 从 0 开始计时，适合课堂活动记录 |
| 播放 / 暂停 | 可点击按钮，也可配合远程按键使用 |

### 语音播报

如果开启语音播报，被抽中的学生姓名会自动读出来。

如果没有声音，按这个顺序检查：

1. 电脑音量是否打开
2. 点名设置里的语音播报是否开启
3. 学校电脑当前输出设备是否正确

### 远程点名键

你可以设置一个远程点名键，默认常见为 `Tab`。

适合的场景：

- 使用翻页笔或无线演示器
- 老师站在讲台上，不想来回走到电脑前

## 7. 画笔与白板

### 开启画笔

点击启动器，选择「画笔 / 白板」后，屏幕上会出现一个半透明工具条。

这时可以直接在屏幕上书写、标注、圈画重点。

### 工具条常用功能

| 功能 | 作用 |
|------|------|
| 毛笔 / 书法笔 | 适合板书和强调 |
| 普通笔 | 适合日常批注 |
| 橡皮 | 擦除笔迹 |
| 区域截图入白板 | 拖拽截取屏幕局部，立即进入白板讲评 |
| 颜色 | 切换笔颜色 |
| 形状 | 画直线、矩形、箭头 |
| 清空 | 清除当前页笔迹 |
| 保存 | 保存当前笔迹截图 |

### 什么时候适合用白板

白板模式适合：

- 数学推导
- 语文板书
- 临时记录学生回答
- PPT 之外的自由讲解

### 笔迹保存与回放

- 每次课的笔迹可以按设置自动保存
- 如果是在图片或 PDF 上书写，笔迹通常会跟随当前页面保存
- 下次上课可以回放或继续使用已保存内容

### 使用提醒

- 先试写两笔，确认触控或手写板输入正常
- 正式上课前确认橡皮和清空功能可用
- 重要内容建议及时保存，不要完全依赖临时状态

## 8. 图片与 PDF 讲解

### 打开文件

1. 点击启动器，选择「打开图片」
2. 选择图片或 PDF 文件
3. 文件会全屏显示，便于投屏讲解

### 常见操作

| 操作 | 结果 |
|------|------|
| 鼠标滚轮上 / 下 | 上一页 / 下一页 |
| 两指捏合 | 缩放图片 |
| 拖拽 | 平移查看细节 |
| 右键 | 关闭当前文件并返回 |

### 适合的课堂场景

- 试卷讲解
- 地图标注
- 图表分析
- 作文素材展示
- 题目圈画

### 如果有很多图片

可以使用图片管理器连续切换。

常见用法：

1. 打开图片所在文件夹
2. 在左侧选择目录
3. 在右侧点缩略图切换内容

## 9. PPT / WPS 演示控制

程序会自动检测正在放映的 PowerPoint 或 WPS 演示文稿。

### 最常见用法

1. 先打开 PPT 或 WPS 并进入放映
2. 再打开画笔功能
3. 直接在幻灯片上批注
4. 需要时通过滚轮或按键翻页

### 常见控制方式

| 操作 | 结果 |
|------|------|
| 在画笔层滚轮上 / 下 | 前进或后退一页 |
| 键盘方向键 | 翻页 |

### 放映状态说明

- 如果 PPT 正在全屏放映，画笔层会尽量贴合放映画面
- 如果不是全屏，画笔层会保持独立，不影响正常编辑
- 如果切换显示器或投影设置，建议重新检查一遍批注位置是否准确

## 10. 快捷键速查表

| 场景 | 按键 | 功能 |
|------|------|------|
| 点名窗口 | `空格` | 触发随机点名 |
| 点名窗口 | `Tab`（可自定义） | 远程点名 |
| 画笔模式 | `Esc` | 退出画笔模式 |
| 图片浏览 | 鼠标滚轮 | 翻页 |
| 图片浏览 | 右键 | 关闭当前图片 |
| PPT 放映中 | 鼠标滚轮 | 翻页 |

## 11. 典型课堂用法

### 课前准备

1. 打开程序，确认悬浮按钮出现
2. 检查班级和学生是否正确
3. 准备好本节课的 PPT、图片或 PDF

### 课堂互动

1. 打开点名窗口
2. 按 `空格` 随机点名
3. 需要限时活动时切换到计时器模式

### 讲解演示

1. 开始 PPT 放映
2. 打开画笔
3. 在幻灯片上直接批注
4. 用滚轮翻页继续讲解

### 习题讲评

1. 打开图片或 PDF
2. 用画笔圈画重点
3. 放大局部细节
4. 翻页后继续讲解

## 12. 常见问题与排查

### 悬浮按钮没有出现

按这个顺序检查：

1. 程序是否真的已经启动
2. 是否被其他窗口挡住
3. 是否跑到了屏幕边缘或副屏
4. 重启程序后是否恢复

如果仍然没有出现，请联系信息老师检查运行环境。

### 点名窗口读不到学生

重点检查：

1. `students.xlsx` 是否存在
2. 工作表名是否是班级名
3. 表格内容是否为空
4. 文件是否正在被其他程序锁定

### 学生照片不显示

重点检查：

1. `student_photos/` 是否存在
2. 班级文件夹名是否与工作表名一致
3. 照片文件名是否使用学号
4. 图片格式是否为常见格式

### 点名有画面但没有声音

按这个顺序检查：

1. 电脑音量
2. 音箱或一体机声音是否正常
3. 程序里语音播报是否开启
4. 当前输出设备是否被切到了别的设备

### 画笔写不出来

重点检查：

1. 是否已经进入画笔模式
2. 当前是否选中了橡皮
3. 触控屏或手写板是否连接正常
4. 是否被其他全屏窗口拦截输入

### PPT / WPS 上无法正常批注或翻页

重点检查：

1. 演示文稿是否已经处于放映状态
2. 当前是不是全屏放映
3. 是否切换过投影模式、分辨率或缩放比例
4. 重开一次放映和画笔后是否恢复

### 笔迹保存在哪里

笔迹通常保存在画笔设置里指定的路径中。  
建议把保存路径设在学校电脑上稳定、固定、容易备份的本地目录。

## 13. 哪些问题需要找信息老师

下面这些问题建议直接交给学校信息老师处理：

- 程序无法启动
- 系统提示缺少 `.NET` 运行时
- 被安全软件或权限策略拦截
- 投影、多屏、分辨率或触控驱动异常
- 学校电脑没有音频输出或外接设备异常

教师自己更适合处理的是：

- 学生名单内容
- 学生照片整理
- 课堂功能切换
- 快捷键和语音播报等常规设置

> 如果实际界面和本文档不完全一致，优先以学校当前部署版本为准；如需反馈问题，再联系项目主页或校内信息支持人员。
